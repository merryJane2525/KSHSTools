# 대표 장비 이미지

첫 페이지 "대표 장비" 카드에 표시할 이미지를 이 폴더에 넣으세요.

## 파일 이름 (필수)

| 파일명 | 표시 위치 |
|--------|-----------|
| `sem.jpg` | SEM |
| `ft-ir.jpg` | FT-IR |
| `freeze-dryer.jpg` | 동결건조기 |

## 권장 사항

- **형식**: JPG, PNG, WebP (코드 기본값은 `.jpg`. PNG/WebP 사용 시 `FeaturedEquipments.tsx`의 `imagePath`를 해당 확장자로 수정)
- **비율**: 가로로 긴 이미지(예: 16:9)가 카드에 잘 맞습니다.
- **크기**: 가로 800px 내외면 충분합니다.

파일을 넣은 뒤 페이지를 새로고침하면 이미지가 표시됩니다. 파일이 없으면 아이콘 플레이스홀더가 보입니다.
